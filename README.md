## 特色

* GCC + NASM 实模式（16 位兼容的 32 位代码）；
* 使用 Shell 脚本自动编译、链接、整合；
* 未经美化的界面
* 软盘镜像仅在 VMware Workstation Pro 15 中测试过；
* 详细的实验报告文档。

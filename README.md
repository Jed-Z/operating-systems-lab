## 说明

* GCC + NASM 实模式（16 位兼容的 32 位代码）；
* Bash 脚本自动编译、链接、整合（非 Makefile）；
* 代码非常普通，功能十分朴素；
* 报告文档特别详细；
* 成绩—时间性价比较高。
